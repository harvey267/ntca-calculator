## **If You Get Errors**

### **Option 1: Virtual Environment (Recommended)**
```bash
# Create clean environment
python -m venv ntca_env

# Activate (Windows)
ntca_env\Scripts\activate

# Activate (Mac/Linux)  
source ntca_env/bin/activate

# Install
pip install streamlit pandas numpy python-dateutil reportlab openpyxl

# Run
streamlit run app.py
```

### **Option 2: Force Reinstall**
```bash
pip install --upgrade --force-reinstall streamlit pandas numpy python-dateutil reportlab openpyxl
```

### **Option 3: Requirements.txt (Pro)**
Create `requirements.txt`:
```
streamlit==1.38.0
pandas==2.2.2
numpy==2.1.1
python-dateutil==2.9.0
reportlab==4.2.2
openpyxl==3.1.2
```

Then:
```bash
pip install -r requirements.txt
```
